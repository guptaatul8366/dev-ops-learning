# Virtualization and VM Setup

This section introduces virtualization concepts and practical VM setup
used for DevOps learning and practice.

---

## Topics Covered

### 1. What is Virtualization
- Virtualization allows running multiple operating systems on a single machine
- Helps create isolated environments for testing and development
- Core concept behind cloud and DevOps infrastructure

### 2. Virtual Machine (VM) Basics
- A VM simulates a physical computer
- Uses a hypervisor to manage resources
- Examples: VirtualBox, VMware

### 3. VM Setup – Manual Method
- Manual installation of Virtual Machines
- Applicable for:
  - Windows
  - macOS (Intel chip)
- Helps understand VM internals and configuration

### 4. VM Setup – Automatic Method
- Automated VM creation using tools
- Faster and repeatable setup
- Reduces manual configuration errors

### 5. VM on macOS M1 Chip
- ARM-based architecture considerations
- Different setup approach compared to Intel-based systems

### 6. Vagrant & VirtualBox Quiz
- Reinforced understanding of virtualization tools
- Covered VM automation basics

### 7. Outro
- Summary of virtualization concepts and VM setup

---

## Key Takeaways
- Virtualization is the foundation of DevOps infrastructure
- VMs provide isolated and consistent environments
- Automation is preferred over manual VM creation
